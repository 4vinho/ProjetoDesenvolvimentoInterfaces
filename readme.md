# Armazém do Seu Zé (versão digital)

Portal front-end para compradores acompanharem catálogo, carrinho e pedidos com o bom humor do Seu Zé.

## Estrutura
- `index.html`: landing page do cliente com destaques e pedidos recentes.
- `catalogo.html`: catálogo interativo com carrinho persistente no navegador.
- `pedidos.html`: histórico dos pedidos feitos pelo comprador.
- `login.html`: acesso para salvar carrinho e dados de contato.
- `registro.html`: criação de conta com validação básica.
- `css/styles.css`: tema escuro, layout responsivo e componentes de cartões/carrinho.
- `js/app.js`: lógica de catálogo, carrinho, pedidos, autenticação simples e persistência em `localStorage`.

Abra qualquer arquivo HTML no navegador para testar. Os dados são mantidos localmente pelo `localStorage`.
